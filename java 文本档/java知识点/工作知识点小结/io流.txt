字符流和字节流：
   字节流可以操作所有类型的文件，因为所有的文件在计算机都是以字节形式存储；而字符流只能用来操作纯文本类型的文件，不能操作字节类型的文件，尽管能复制到图片到另一个文件夹中但都是已经损坏的图片。
字节流和字符流的区别：
   字节流操作的基本单元为字节；字符流操作的基本单元为Unicode码元；
   字节流默认不使用缓冲区；字符流使用缓冲区
   字节流通常用于处理二进制数据，实际上它可以处理任意类型的数据，但它不支持直接写入或读取
   字符流通常处理文本数据，它支持写入及读取Unicode码元。
图片不能正常显示的原因：
   因为字符流是读取字节后，缓存，然后去码表查找匹配，若是匹配不到就会到位置码表位置寻找类似的，此时返回的数据就可能不正确。因此生成的新图片的编码和老图片的编码是不一致的，从而导致图片不能正常显示。

问题：字符流可以追加字符串但是会导致图片无法显示，但是字节流无法追加。但是却不会影响图片显示。
